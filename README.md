# medien-sammel-app
App to visually map out how you connect different pieces of media.

## Setup (Vite + React PWA)

- Install dependencies:

```bash
npm install
```

- Run dev server:

```bash
npm run dev
```

- Build for production:

```bash
npm run build

# preview the production build
npm run preview
```

This scaffold uses `vite-plugin-pwa` to generate a service worker and manifest. The dev server serves the `public/manifest.webmanifest` during development.
